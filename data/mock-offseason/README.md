# Mock Offseason Data Directory

This directory stores all data for the Mock Offseason system.

## Files

- `league-settings.json` - Main league configuration and settings
- `transactions/` - Trade and signing transaction logs
- `teams/` - Individual team data (roster, picks, cap)
- `players/` - Player database
- `draft/` - Draft prospects and results

## Notes

- All data is stored per-guild (server)
- Data is NOT shared between servers
- This system is completely separate from PATS and other bot features
